Question-1: What are the different ways to select an element in the DOM?

The DOM provides various ways to select elements: by ID, class name, tag name, CSS selectors, or name attribute. Methods include `getElementById`, `getElementsByClassName`, `getElementsByTagName`, `querySelector`, `querySelectorAll`, and `getElementsByName`.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

`innerHTML` returns or sets the HTML content inside an element, including tags. `innerText` shows the visible text only, while `textContent` shows all text, even hidden, without formatting.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to handle events on its child elements. It works by using event bubbling, allowing efficient handling of many dynamic child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the target element and moves upward through its parent elements in the DOM tree. This allows parent elements to react to events triggered on their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

You can create elements using `document.createElement()`, add them with `appendChild()` or `append()`, and remove them using `removeChild()` or `element.remove()`. These methods allow dynamic changes to the DOM structure.
